# Aisha Cannes Showcase

Luxury cinematic web showcase for **Aisha and the Sands of Destiny** by BrightBox Animations.

## Run Locally

```bash
cd "/Users/vaquita/Downloads/aisha/aisha-cannes-showcase"
npm run dev -- --host 0.0.0.0
```

Local URL:

```text
http://localhost:5173/
```

## Build For Hosting

```bash
npm run build
```

The deployable static build is created in:

```text
/Users/vaquita/Downloads/aisha/aisha-cannes-showcase/dist
```

## Asset Policy

Original Aisha files are not edited. The website uses copied assets under:

```text
/Users/vaquita/Downloads/aisha/aisha-cannes-showcase/public/assets
```

Current asset groups:

- `trailers/`
- `music/`
- `characters/`
- `locations/`
- `props/`
- `documents/`
- `style/`
